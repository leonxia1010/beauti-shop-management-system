# Beauty Shop Management System - ASCII 布局设计

## 🎨 设计系统规范

### 颜色系统

```scss
// 主色调
$primary: #14b8a6; // 青绿色 - 专业可信
$secondary: #0f172a; // 标题、正文深色
$success: #10b981; // 成功状态
$warning: #f59e0b; // 警告
$danger: #ef4444; // 错误/异常
$neutral-bg: #f8fafc; // 背景
$neutral-border: #e2e8f0; // 边框
$neutral-text: #64748b; // 次级文本
```

### 字体规范

- H1: 24px / 600 / 32px line-height
- H2: 20px / 600 / 28px line-height
- H3: 18px / 500 / 26px line-height
- Body: 14px / 400 / 22px line-height
- Small: 12px / 400 / 18px line-height

### 间距系统

- 基础步长: 4px
- 常用间距: 4/8/12/16/24/32px
- Web 网格: 12 列，间距 16px
- 移动端: 4 列网格

## 🎨 组件规范总结

### Web 端核心组件

- **KPI Summary Card**: 关键指标卡片，颜色对比度 ≥7:1
- **TrendChart**: 收入趋势图 (Tremor + Recharts)
- **Batch Card**: 现金交接批次卡片，差异金额突出显示
- **Kanban Column**: 异常任务看板列
- **Stepper Form**: 三步导入流程表单
- **Editable Table**: 支持 inline 编辑的数据表格
- **Status Tabs**: 状态标签切换
- **Timeline**: 审批时间线组件

### 移动端核心组件

- **Task Card**: 待办任务卡片
- **Quick Action**: 大按钮快捷操作（44×44px）
- **Segmented Form**: 分段式表单
- **Offline Indicator**: 离线状态指示器
- **Success Toast**: 成功提示（250ms 淡入淡出）
- **浮动标签**: 输入框的动态标签

### 动画时长

- 快速交互: 200ms
- 一般过渡: 300ms
- 复杂动画: 500ms
- 缓动函数: cubic-bezier(0.4, 0, 0.2, 1)

### 响应式断点

- Mobile: < 768px（单列）
- Tablet: 768px - 1023px（两列）
- Desktop: 1024px - 1439px（12 列网格）
- Wide: ≥ 1440px（宽屏优化）
